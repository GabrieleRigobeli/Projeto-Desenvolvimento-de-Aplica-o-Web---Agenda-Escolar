# Agenda Escolar

Projeto desenvolvido como parte dos estudos de **Desenvolvimento de Aplicações Web**. Trata-se de uma ferramenta para auxiliar na organização de rotinas estudantis e controle de prazos.

## Tecnologias Utilizadas
* **HTML5**: Estruturação semântica.
* **CSS3**: Estilização e layout responsivo.
* **JavaScript**: Lógica de interação (se aplicável ao projeto atual).
